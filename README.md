# Projekt Zbiorniki – Symulacja Kaskady

Aplikacja desktopowa napisana w Pythonie z wykorzystaniem PyQt5, 
przedstawiająca symulację przepływu cieczy pomiędzy zbiornikami 
połączonymi rurami, zaworem, pompą oraz grzałką.

## Funkcjonalności

- Symulacja przepływu cieczy pomiędzy 4 zbiornikami
- Rury z animacją przepływu
- Zawór sterujący przepływem między zbiornikami
- Pompa zwiększająca prędkość przepływu (ON/OFF + suwak)
- Grzałka podnosząca temperaturę cieczy
- Alarm po osiągnięciu 100°C
- Ręczne napełnianie i opróżnianie zbiorników
- Wizualizacja poziomu cieczy w zbiornikach

## Uruchomienie

 Aby uruchomić symulację zbiorników, wykonaj poniższe kroki:

1. Zainstaluj Python 3.7 lub nowszy.
2. Zainstaluj PyQt5.
3. Przejdź do katalogu projektu cmd: cd ścieżka\do\projektu
4. Uruchom symulacje: python main.py
